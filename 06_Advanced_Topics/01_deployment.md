# 模型部署簡介

## 部署方式

| 方法         | 說明                       |
|--------------|----------------------------|
| Flask API    | 提供 HTTP 服務              |
| FastAPI      | 輕量、高效、async 支援       |
| Docker       | 環境封裝，方便移植與佈署     |
| Streamlit    | 快速製作互動式 Web App       |

## 基本流程

1. 儲存模型 (`torch.save`)
2. 載入模型並建立 API
3. 測試與上線（本地/雲端）
